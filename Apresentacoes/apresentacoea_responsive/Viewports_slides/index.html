Os 3 tipos de pixels

  • • CSS pixels
  • • Device pixels
  • • Density-independent pixels

CSS Pixels

São aqueles usados nas declarações CSS como:

div {
  width: 190px;
  padding: 20px;
}

Device pixels

São os pixels físicos do device, seu valor é fixo e depende do device.

Device pixels

Mas os devices mais novos aparecem com cada vez mais desidade de pixels.

Em teoria a largura da viewport também deveria aumentar.

Mas isto poderia quebrar sites e aplicativos

Density-independent pixels (dips)

Os fabricantes criaram pixels que independem da desidade.

O número de "dips" é igual ao número de CSS pixels ótimo para a visualização a 100%

Qual é o tipo de pixel?

Se uma propriedade Javascript é expressa em pixel, pergute-se qual é o tipo de pixel.

Geralmente é CSS pixels para coisas relacionadas ao DOM.

Algumas vezes é dips ou device pixels para coisas relacionadas a tamanho de tela.

Os 2 tipos de viewports

  • • Layout viewport
  • • Visual viewport

A viewport é a quantidade de espaço disponível para o layout CSS.

No desktop, é igual à janela do navegador.

No mobile é diferente:

Navegadores mobile precisam renderizar corretamente todos, os sites, mesmo aqueles que não foram otimizados para mobile

Nomalmente o device tenta encaixar o site em uma largura de 960px
